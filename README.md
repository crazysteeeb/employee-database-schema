# Employee Database Schema

A relational SQL database project demonstrating schema creation, foreign key relationships, data population, and query implementation.

## Overview

This project models an employee-company management system using relational database concepts.

The database includes:
- Employees
- Companies
- Employment relationships
- Management hierarchies

## Features

- Relational schema design
- Primary and foreign key constraints
- Data population scripts
- SQL query implementation
- Relationship modeling

## Technologies Used

- SQL
- Relational Databases

## Database Tables

- employee
- company
- works
- manages

## Concepts Demonstrated

- Relational database design
- Entity relationships
- Foreign key constraints
- SQL joins
- Query filtering and sorting
- Data modeling

- Designing normalized relational schemas

- Writing SQL queries for relational databases

- Structuring database systems logically
